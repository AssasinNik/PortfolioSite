NewsApp

Android app to get actual Russian news from API

About the Project

This Android application allows users to get different news from different Russian news sources. Users can filter them with categories

Technologies

  • UI Framework: Jetpack Compose
  • Database: Room
  • Architecture: MVI Pattern
  • Concurrency: Kotlin Coroutines
  • Networking: Retrofit/OkHttp
  • Dependency Injection: Dagger Hilt
  • Image Loading: Coil
