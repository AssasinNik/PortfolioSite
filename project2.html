Films for Mood

Android app to help you choose movies based on your mood

About the Project

This Android application allows users to select films based on their mood. It uses the Film API Server to fetch film data and implements features like trending and newest films.

Technologies

  • UI Framework: Jetpack Compose
  • Database: Room
  • Architecture: MVVM/MVI Patterns
  • Concurrency: Kotlin Coroutines
  • Networking: Ktor
  • Dependency Injection: Dagger Hilt
  • Image Loading: Coil
