Marvel Library

Android app to help find information about Marvel

About the Project

This Android application allows users to find a lot of different information about Marvel Comics, Films, TvShows, Heroes. They also can watch trailers of their favourite films. And get more fun about popular heroes.

Technologies

  • UI Framework: Jetpack Compose
  • Database: Room
  • Architecture: MVVM Pattern
  • Concurrency: Kotlin Coroutines
  • Networking: Retrofit/OkHttp
  • Dependency Injection: Dagger Hilt
  • Image Loading: Coil
  • Logger: Timber
